项目说明
《步步生花》是一款基于RenPy开发的单机宫斗文字互动游戏，延用旧版深宫曲制作而成
# 核心教程：下载项目+RenPy打开
1. 必备工具：RenPy安装（必装）
- 下载地址：https://www.renpy.org/latest.html
- 下载对应系统版本（Windows/Mac/Linux），安装过程默认下一步即可，无需额外配置。
- 补充说明：RenPy为免费开发引擎，支持商用与非商用，安装后无需额外插件。
2. 项目下载
前往指定项目下载地址，下载《步步生花》单机版项目压缩包（格式为.zip/.7z），下载完成后，解压至电脑任意路径（建议路径无中文、无特殊符号，避免打开失败）。
3. 用RenPy打开项目
1. 启动已安装的RenPy软件，进入主界面。
2. 点击主界面“Load Project”（加载项目），在弹出的窗口中，找到步骤2中解压后的项目文件夹，选中文件夹并点击“选择”。
3. 项目加载完成后，点击RenPy主界面的“Launch Project”（启动项目），即可开始游玩。
常见问题
- 问题：项目无法加载/启动失败？
解决：检查RenPy版本是否兼容，确保项目解压路径无中文、无特殊符号，重新解压项目后再次尝试加载。
- 问题：启动后无画面/卡顿？
解决：关闭电脑后台多余程序，确保设备存储空间充足（预留100MB以上）。
